# 桌面应用-星空3

## 一、技术栈

vite2 + vue3 + electron19 + lowdb

> 开发环境
* node v16.15.1
* yarn 1.22.15
* npm 8.11.0

```sh
// 下载代码
git clone https://github.com/qypone/vue-electron-template.git
或 git clone git@github.com:qypone/vue-electron-template.git

// 导入依赖
yarn

// 运行项目
yarn run electron:serve

其它脚本详见package.json
```
> 参考文档 /doc/about_tech.md
