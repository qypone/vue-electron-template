# Element

## 安装element
```
yarn add element-plus
```
## 使用element
* main.js中引入
```
import ElementPlus from 'element-plus';
import 'element-plus/dist/index.css';

app.use(ElementPlus)
```
* .vue文件中使用
```
<el-button type="success">Success</el-button>
```
